### Installation:

#### Install Ruby:
https://rubyinstaller.org/downloads/

#### Check Ruby version:
`gem --version`

#### Install Rails:
`gem install rails`

#### Check Rails version:
`rails -v`

#### Install database:
`gem install sqlite3 -v '1.4.1' --source 'https://rubygems.org/'`

#### Install all dependencies:
`bundle install`

#### Start server:
`rails server`
